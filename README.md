# Система управления рецептами

Домашнее задание №2 по курсу "Технологии программирования", НИУ ВШЭ 2025-26.

## Описание

Консольная система для создания рецептов, масштабирования порций и генерации списка покупок. Реализованы классы: `Ingredient`, `Recipe`, `DietaryRecipe`, `ShoppingList`.

## Установка

```bash
git clone <ссылка-на-репозиторий>
cd hw2-recipes
pip install -r requirements.txt
```

## Запуск тестов

```bash
pytest test_recipes.py -v
```
